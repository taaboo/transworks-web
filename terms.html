最終更新日：2026年6月10日 ／ 施行日：2026年6月25日

TransWorks について

TransWorks は、多言語チャットアプリです。職場の多言語環境におけるコミュニケーションツールとして設計されています。雇用主や管理者、そして外国人労働者を含むチームメンバーが、日々の業務上のコミュニケーションにおいて言語の壁を越えるためのアプリです。

本規約は、どの言語に翻訳されても読みやすいよう、平明な文章で書いています。ご不明な点がありましたら、info@transworks.net までお問い合わせください。

01

本規約への同意

本利用規約（以下「本規約」）は、東海林忠樹（以下「当方」）が運営する TransWorks アプリ（以下「本アプリ」）の利用条件を定めるものです。

本アプリの初回起動時、ユーザー言語を選択した後、利用者登録の前に本規約が表示されます。本規約に同意することで、本アプリの利用を開始できます。同意できない場合は、本アプリをご利用いただけません。

02

TransWorks の性質

TransWorks は、職場における業務コミュニケーションを支援するツールです。職場の人間関係は、本来、対面でのやり取り——朝礼、昼食時の会話、現場での協働、会議——の中で築かれます。TransWorks は、そうした人間関係の中で生じる共有認識の齟齬を埋めるためのアプリであり、対面コミュニケーションを置き換えるものではありません。

TransWorks は以下のようなアプリではありません：

  • 一般消費者向けの SNS
  • 個人的なメッセージング・サービス
  • マッチング・出会い系サービス
  • エンターテイメント向けのメッセージング・サービス

TransWorks のチャットルームは、雇用主や企業が利用料を負担することが想定される業務空間です。業務目的のコミュニケーションのために設計されています。

  • 登録情報を正確に提供する
  • 登録情報を最新の状態に保つ
  • アカウントの認証情報を第三者に開示・貸与・譲渡しない

18歳未満の方は、保護者の同意を得たうえで本アプリをご利用ください。

04

業務記録の永続性について

本アプリで送受信されたメッセージ、画像、音声、その他のコンテンツ（以下「コンテンツ」）は、業務記録として保存されます。利用者は、送信したメッセージを後から削除することはできません。これは意図的な設計です。

職場でのコミュニケーションは、誤解の解消、指示内容の確認、安全管理の確認、業務上のトラブル対応など、後から参照される必要がしばしば生じます。メッセージを削除できる仕組みは、これらの正当な業務記録の目的を損なうおそれがあります。

本アプリを利用される際は、送信したメッセージが他のルームメンバーや管理者から引き続き閲覧可能であり、業務記録の保全のため当方が保存する場合があることをご理解ください。

この永続性は、TransWorks が業務ツールとして持つ価値の一部です。責任あるコミュニケーションを心がけてください。

06

通報機能と証拠保全

本規約に違反する行為を目撃した場合、本アプリの通報機能を通じて報告することができます。メッセージを通報するには、該当メッセージがあるチャットルームの背景をダブルタップしてください。本アプリは以下を実行します：

  1. 通報対象メッセージのスクリーンショットを自動生成
  2. チャットルームID、対象利用者のユーザーID、送信時刻を含むウォーターマークを付与
  3. スクリーンショットを通報者のフォトライブラリに保存
  4. スクリーンショットを当方の Firestore データベースにアップロード

通報者の端末と当方のデータベース、両方に同じスクリーンショットが保存される仕組みは、職場で発生した事案の証拠を改ざんから守るためのものです。ウォーターマークが偽造を防ぎます。

当方は、職場のトラブル対応、法令上の要請、または利用規約違反の調査が必要な場合に、通報されたコンテンツを確認することがあります。

通報機能は責任を持って使用してください。悪意のある通報や虚偽の通報は、それ自体が本規約違反となる場合があります。

AI 翻訳・処理機能について

本アプリは、翻訳、音声認識（STT）、音声合成（TTS）、画像内テキスト認識（OCR）、AI アシスタントなどの機能を提供するために、OpenAI、Google Gemini、Google ML Kit、Google Vision その他の第三者の AI サービスを利用します。

AI による処理結果の正確性は保証されません。翻訳には誤りが含まれる可能性があります。音声認識や画像認識にも誤認識が生じる場合があります。AI アシスタントの応答にも不正確な内容が含まれる可能性があります。

業務上の重要な判断、法的判断、医療的判断、緊急対応などにおいては、AI 処理の結果のみに依存せず、必要に応じて人間の専門家による確認を行ってください。

AI 処理の不正確性、翻訳エンジンの誤翻訳、音声認識や画像認識の誤認識に起因する損害について、当方は責任を負いません。

本アプリは、機能の利用にポイント制を採用しています。ポイントは App Store のアプリ内課金（IAP）を通じて購入できます。

価格について：ポイントパックの価格は、Apple の地域別価格テーブル（Price Tier）に従って設定されます。利用者の App Store の地域に応じて、現地通貨で表示・課金されます。

消費レート：機能ごとのポイント消費レートは米ドル建てで設定されています。これは、本アプリが利用する外部 API の利用料金が米ドル建てで設定されていることに対応しています。

アカウント削除時の扱い：アカウントを削除した場合、保有するポイントの一部または全部が、当該アカウントが所有するチャットルームの運営のために引き続き使用されること、または消滅することがあります。いずれの場合も、ポイントの払い戻しは行いません。

返金について：当方からのポイントの返金は原則としてお受けできません。ただし、App Store の返金ポリシーに基づく返金請求はこれを妨げません。

消費レートの変更：外部 API の料金改定や運営上の必要により、消費レートを変更する場合があります。重要な変更については、事前にアプリ内で通知します。

ポイント通帳：ポイントに関するすべての取引（購入、消費、付与）は、アプリ内の「ポイント通帳」に記録され、いつでも確認できます。

詳細について：機能ごとの消費ポイントの目安、付与条件、その他の詳細は、本アプリ内の「ポイントについて」画面に記載します。

データの保存期間

当方は、本アプリで取り扱うデータを以下の方針で保存します。

  1. アカウント情報（メールアドレス、ニックネーム、プロフィール画像等の個人を特定する情報）：利用者がアカウント削除を実行した時点で、即時に削除します。
  2. チャットメッセージ：アカウントおよびチャットルームが有効である期間中、業務記録として保存されます。
  3. 画像・ファイル・動画：アカウントおよびチャットルームが有効である期間中、保存されます。
  4. 音声データ：テキストへの変換処理完了後、即時削除されます。変換されたテキストは、チャットメッセージと同様に保存されます。
  5. OCR処理用 PDF ファイル：処理完了後、即時削除されます。
  6. ポイント購入履歴：税法上の必要性に基づき、7年間保存します。
  7. 通報スクリーンショット：対応完了後6ヶ月間保存します。
  8. 職種・勤務国データ：個人を特定できない形に匿名化したうえで、期間の定めなく保存します。

Data Retention Periods

We retain data handled by the App according to the following policy:
